깃 커밋 과정
1. 깃저장소 만들기
깃 커밋을 하기전에 깃 저장소로 쓸 디렉토리를 하나 만듭니다.
mkdir 깃저장소 이름

2. 해당 저장소로 이용해 git init을 입력합니다.
git init을 하는이유는 깃을 사용할수 있도록 디렉토리를 초기화해야하기 때문입니다.
git init을 하면 Initialized empty Git repository 라는 메세지가 나오면서 사용이 가능합니다.

3. 깃을 초기화 할때 만들어지는 .git 디렉토리 안에는 가상의 영역이 존재합니다.
작업트리 -> 스테이지 -> 저장소

(1) 깃디렉토리에서 평범하게 vim hello.txt 파일을만들면  작업트리에서 파일을 만든것입니다.
(2) 이제 아까만든 파일을 스테이지 영역에 파일을 추가합니다. ex) git add hello.txt
(3) 이제 스테이지 영역에 있는 파일을 저장소로 올립니다. 이것을 git commit 이라고 합니다.
